## 3. 구현 컴포넌트

### 로그인 헤더 (common 레이어, 로그인한 모든 페이지 공용)
- **영상생성** `.btn-header-video` — 검정 CTA + `video` 아이콘
- **크레딧** `.header-credit` — surface pill, `coins` 아이콘(레드), `10,908 크레딧`
- **프로필 드롭다운** `.header-profile` — 아바타(흰 배경·검정·보더=기본 프로필) 클릭 → 이름/이메일 + 미구독 배지 + 메뉴(마이페이지/크레딧 충전/로그아웃)
- 동작: `common.js` `initHeaderProfile()` (클릭 토글·바깥클릭·Esc)

### 모바일 드로어 (로그인 상태)
- 헤더 우측: 영상생성 + 햄버거 그룹(`.site-header__mobile`)
- 드로어 순서: 프로필+클로즈(같은 행) → 크레딧(전체폭·18px 볼드·아이콘 22px) → [영상생성 | 마이페이지] 반반 1행 → 기능 nav → 로그아웃
- 로고 모바일 폭 `auto`로 완화

### 로그아웃 확인 모달 (mypage-common 레이어, 3페이지 공용)
- 트리거 `[data-logout-open]`(푸터·드로어·프로필 드롭다운), 모달 `[data-logout-overlay]`
- "로그아웃하시겠습니까?" + [취소 | 로그아웃] 반반, 바텀시트↔중앙카드 반응형
- 동작: `mypage-common.js` `initMypageLogout()`

### 회원탈퇴 페이지 `withdraw.html`
- 경고 아이콘 → 제목(브랜드 레드) → 안내 2블록(비디오/프로젝트, 크레딧 소멸) → 복구불가 경고 → 확인 문구 입력 → [취소 | 회원탈퇴] 반반
- 게이트: 정확히 `탈퇴하겠습니다` 입력 시에만 회원탈퇴 버튼 활성(`withdraw.js`), 확인 시 `index.html` 이동
- mypage/mypage-login 푸터 `회원탈퇴` → 이 페이지로 링크
- 모바일: 좌우 풀블리드 + 헤더 하단 밀착
